For this project we are using Minix version 3.1.8
The package gcc44-4.4.3 must be installed to make this project.
To install it type "pkgin install gcc44-4.4.3"

To install modified kernel:
type "./install"
when the process completes, type "shutdown"
type "menu", select the modified kernel

To build test files:
type "make"
